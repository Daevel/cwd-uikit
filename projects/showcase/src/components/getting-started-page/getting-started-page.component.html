Getting started with CodeWithDaevel Uikit

This guide explains how to set up your Angular project to begin using CodeWithDaevel Uikit. It includes information on prerequisites, installing CodeWithDaevel Uikit, and optionally displaying a sample Uikit component in your application to verify your setup.

This guide assumes that the Angular CLI has already been installed.

Install CodeWithDaevel Uikit

Add CodeWithDaevel Uikit to your application by running the following command:

npm install @daevel/uikit
Display a component

Let's display a button component in your app and verify that everything works.

You need to import the CwdButtonComponent that you want to display by adding the following lines to your standalone component's imports, or otherwise your component's NgModule.


    import { CwdButtonComponent } from 'cwd/uikit';
    @Component ({
    imports: [
        CwdButtonComponent,
    ],
    })
    class AppComponent {}

Add the cwd-button tag to the app.component.html like so:


    <cwd-button>Click me!<cwd-button>

Run your local dev server:


    npm start

Then point your browser to http://localhost:4200

You should see the Uikit button component on the page.

In addition to the installation schematic, CodeWithDaevel Uikit comes with several other schematics (like nav, table, address-form, etc.) that can be used to easily generate pre-built components in your application.
